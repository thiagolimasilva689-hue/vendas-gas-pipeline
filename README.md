# 💨 Pipeline de Vendas de Gás

Pipeline ETL com dados reais de vendas de gás.

## 📋 Funcionalidades
- Importação de CSV com dados reais de vendas
- Consultas SQL: faturamento por dia, por forma de pagamento
- Identificação de clientes que compraram fiado
- Exportação para Excel com análises

## 🛠️ Tecnologias
- Python 3
- SQLite3
- CSV
- Excel

## 📁 Arquivos
| Arquivo | Descrição |
|---------|-----------|
| `vendas_gas_07.csv` | Dados reais de vendas |
| `importar.py` | Código Python com pipeline e consultas |
| `README.md` | Este arquivo |

## 🔍 Exemplos de Consultas
- Total de vendas por dia
- Forma de pagamento mais usada (PIX)
- Clientes que compraram fiado
- Média de preço do botijão
